# 二. 环境搭建

⏹安装脚手架工具

`npm install -g @angular/cli@7.0.6`

通过`ng v` 命令查看是否安装成功

⏹通过脚手架项目创建

`ng new 项目名称`

<img src="img/image-20211122215213647.png" alt="image-20211122215213647" style="zoom:80%;float:left" />

⏹运行项目

`ng serve --open`
